# cpp_ros_su26

This repo is for learning purposes and exploring C++ in a ROS context. All resources can be found at [learncpp.com](https://www.learncpp.com).

## Setup

If cloning this repo onto a new machine, run the following command to set up `clangd`:

```bash
cmake -S . -B build -DCMAKE_EXPORT_COMPILE_COMMANDS=ON
```

## Build

To compile the executable, run the following command:

```bash
cmake --build build
```
